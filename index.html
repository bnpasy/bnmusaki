Struktur Dasar Algoritma & Notasi Algoritmik

Kerangka penyelesaian masalah secara sistematis.

Struktur dasar algoritma adalah pola yang digunakan untuk menyusun langkah-langkah penyelesaian masalah. Terdapat tiga struktur dasar yaitu Sekuensial, Seleksi, dan Perulangan.

a. Sekuensial (Urutan)

Instruksi dijalankan secara berurutan dari atas ke bawah.

Contoh Algoritma 
1. Masukkan panjang
2. Masukkan lebar
3. Hitung luas = panjang × lebar
4. Tampilkan luas

b. Seleksi (Percabangan)

Digunakan untuk memilih tindakan berdasarkan kondisi tertentu. 

if nilai >= 75
    Maka "Lulus"

else:
   Maka "Tidak Lulus"

c. Perulangan (Looping)

Digunakan untuk mengulang proses beberapa kali. 

Untuk i = 1 sampai 5
    Tampilkan "Hello"
2. Notasi Algoritmik (Skema)

Notasi algoritmik adalah cara menuliskan algoritma agar mudah dipahami manusia sebelum diterjemahkan ke dalam bahasa pemrograman.

Contoh Notasi Algoritmik 
Algoritma Luas Persegi Panjang

Deklarasi
panjang, lebar, luas : integer

Deskripsi
Input panjang
Input lebar
luas ← panjang × lebar
Output luas
Judul Algoritma

Nama algoritma yang dibuat.

Deklarasi Variabel

Daftar variabel yang digunakan.

Deskripsi

Langkah-langkah penyelesaian masalah.

Tipe Data Keterangan Contoh

Ekspresi adalah gabungan nilai, variabel, dan operator yang menghasilkan suatu nilai.

Ekspresi Aritmatika

Ekspresi Relasional

umur <= 18

Ekspresi Logika

A and B

not A

Harga atau value adalah nilai yang dimiliki suatu variabel.

03

Struktur Algoritma Sekuensial

Instruksi berurutan tanpa percabangan dan tanpa perulangan.

Instruksi pada algoritma adalah perintah atau langkah-langkah yang harus dijalankan secara sistematis untuk menyelesaikan suatu masalah.

Struktur sekuensial adalah struktur algoritma yang dijalankan secara berurutan dari atas ke bawah tanpa percabangan dan tanpa perulangan.

Ciri-ciri Struktur Sekuensial
📌
Berurutan
Langkah dikerjakan dari atas ke bawah.
🚫
Tanpa Percabangan
Tidak menggunakan IF atau ELSE.
🔁
Tanpa Perulangan
Tidak menggunakan loop.
1️⃣
Satu Kali Eksekusi
Setiap instruksi dijalankan satu kali.
📖
Mudah Dipahami
Alur sederhana dan linear.
Contoh Struktur Sekuensial

Menghitung Keliling Persegi 

1. Mulai
2. Masukkan panjang sisi (S)
3. Hitung keliling

   K = 4 × S

4. Tampilkan K
5. Selesai
📝 Tugas Sekuensial
Notasi Deskriptif
  1. Mulai.
  2. Masukkan nama pembeli.
  3. Masukkan judul film.
  4. Masukkan jumlah tiket yang dibeli.
  5. Masukkan harga satu tiket.
  6. Hitung total harga = jumlah tiket × harga tiket.
  7. Masukkan jumlah uang yang dibayarkan.
  8. Hitung kembalian = uang bayar − total harga.
  9. Tampilkan struk.
  10. Selesai.
Pseudocode 
Mulai

    Input nama_pembeli
    Input judul_film
    Input jumlah_tiket
    Input harga_tiket

    total_harga = jumlah_tiket * harga_tiket

    Input uang_bayar

    kembalian = uang_bayar - total_harga

    Output "===== STRUK PEMESANAN ====="
    Output nama_pembeli
    Output judul_film
    Output jumlah_tiket
    Output total_harga
    Output uang_bayar
    Output kembalian

Selesai
Kode Program Python
bioskop.py 
print("===== PEMESANAN TIKET BIOSKOP =====")

nama = input("Nama Pembeli : ")
film = input("Nama Film : ")

harga = int(
input("Harga Tiket : ")
)

jumlah = int(
input("Jumlah Tiket : ")
)

total = harga * jumlah

bayar = int(
input("Uang Bayar : ")
)

kembalian = bayar - total

print("\n===== STRUK =====")

print("Nama :", nama)
print("Film :", film)
print("Total :", total)
print("Bayar :", bayar)
print("Kembalian :", kembalian)

Perulangan (Looping)

Perulangan digunakan untuk menjalankan instruksi yang sama secara berulang.

1. Perulangan FOR

Perulangan FOR digunakan ketika jumlah pengulangan sudah diketahui sebelumnya. 

total = 0

for i in range(1, 6):
    nilai = int(
    input(f"Masukkan nilai mahasiswa ke-{i}: "))

    total += nilai

print("Total nilai =", total)
Output: 
Masukkan nilai mahasiswa ke-1: 80
Masukkan nilai mahasiswa ke-2: 75
Masukkan nilai mahasiswa ke-3: 90
Masukkan nilai mahasiswa ke-4: 85
Masukkan nilai mahasiswa ke-5: 70

Total nilai = 400
2. Perulangan WHILE

Digunakan ketika jumlah pengulangan belum diketahui dan bergantung pada kondisi. 

angka = 5

while angka > 0:
    print(angka)
    angka -= 1

print("Selesai")
Output: 
5
4
3
2
1

Selesai
3. Perulangan DO-WHILE

Python tidak memiliki do-while, sehingga biasanya dibuat menggunakan while True. 

while True:

    pilihan = input("Ketik Y untuk lanjut : ")

    if pilihan == "Y":
        print("Program dilanjutkan")
        break

    else:
        print("Input salah")
4. Nested Loop

Nested Loop adalah perulangan di dalam perulangan. 

for i in range(1,6):

    for j in range(1,6):

        print(i*j,end="\t")

    print()
Output: 
1  2  3  4  5
2  4  6  8 10
3  6  9 12 15
4  8 12 16 20
5 10 15 20 25
FOR

WHILE
Berdasarkan kondisi
DO-WHILE

NESTED LOOP

Array (larik) adalah struktur data yang digunakan untuk menyimpan banyak data dalam satu variabel.

Macam-Macam Array

Array dibedakan berdasarkan jumlah dimensinya.

1. Array 1 Dimensi (1D)

Array yang hanya memiliki satu baris dan menggunakan satu indeks.

Python
angka = [10, 20, 30, 40, 50]

print(angka[0])   # 10
print(angka[3])   # 40

Ilustrasi

Indeks : 0   1   2   3   4
Data   :10  20  30  40  50

2. Array 2 Dimensi (2D)

Array yang terdiri dari baris dan kolom.

Python
matriks = [[1,2,3],[4,5,6],[7,8,9]]

print(matriks[0][1])   #2
print(matriks[2][0])   #7

Ilustrasi

      Kolom
      0  1  2

Baris

0   [1,2,3]
1   [4,5,6]
2   [7,8,9]

3. Array Multidimensi (3 Dimensi atau lebih)

Array yang memiliki lebih dari dua dimensi.

Python
data = [[[1,2],[3,4]],[[5,6],[7,8]]]

print(data[0][1][0])   #3
print(data[1][0][1])   #6

Ilustrasi

Dimensi 1
├── Dimensi 2
│   ├── [1,2]
│   └── [3,4]
└── Dimensi 2
    ├── [5,6]
    └── [7,8]
Ringkasan
Jenis Array Bentuk Contoh
1 Dimensi Satu baris [10,20,30]
2 Dimensi Baris dan kolom [[1,2,3],[4,5,6]]
Multidimensi Lebih dari dua dimensi [[[1,2],[3,4]],[[5,6],[7,8]]]
